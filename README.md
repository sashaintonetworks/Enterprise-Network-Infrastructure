# Enterprise Network Infrastructure
Repo pubblica contenente un file .pkt che contiene la progettazione logica di un'infrastruttura di rete enterprise HA, seguito da vari file testuali dove sono riportate le spiegazioni, le scelte tecniche, i protocolli, le misure di sicurezza ecc... adottate in questa infrastruttura. Successivamente, al caricamento completo di tutti i file testuali citati, seguira la realizzazione su software professionale EVE-NG per simularne il reale funzionamento dell'infrastruttura, usando configurazione esclusivamente da CLI e immagini di sistema reali dei dispositivi di rete (sw, router, ap, fw ecc...)
